## 📋 Sobre

Aplicação web do Zentrax, consumindo a API REST para exibir e gerenciar dados com interface responsiva e moderna.

## 🚀 Como Rodar

**Pré-requisitos:** Node.js 18+ e [API do Zentrax](https://github.com/LuanMSouza/zentrax-back) rodando.

```bash
git clone https://github.com/LuanMSouza/zentrax-front
cd zentrax-front
npm install
cp .env.example .env
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).
